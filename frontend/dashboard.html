Workflow guide

Workflow

Choose Python or Java, submit source, review compiler diagnostics, then inspect security, quality and remediation tabs.

Architecture

HTML, CSS and JavaScript provide the workspace. FastAPI validates source and coordinates the existing analysis services.

FastAPI, Python AST validation, javac, ChromaDB, JavaScript, Chart.js and jsPDF.

Future scope

Team workspaces, CI integration, repository scans and policy-driven security gates.
